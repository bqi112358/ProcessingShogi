# ProcessingShogi
・Processing3(Python)で遊べる将棋プログラム  
・ポンコツAI付き!!!  


## AIについて
・探索アルゴリズムは NegaMax法 + αβ枝刈り  
・静的評価関数は駒割りのみ  
・レベル（探索深度）は1~3で変更可能
